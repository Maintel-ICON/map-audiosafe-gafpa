Audiosafe Help Guide

Audiosafe Call Recording

What is Audiosafe?

Audiosafe, powered by the Maintel Application Platform (MAP), provides secure, long-term storage for call recordings and metadata from cloud-based contact centre platforms such as Genesys Cloud. It helps customers meet regulatory requirements and retain valuable data for business insight and compliance purposes.

Key Features and Benefits

  • FCA-compliant, long-term call recording storage
  • Search, filter, and playback functionality
  • Metadata retention beyond vendor-imposed time limits
  • Secure AWS-based storage with encryption at rest and in transit
  • Role-based access and Single Sign-On (SSO) via Microsoft Entra
  • Ongoing platform monitoring and alerting

How Audiosafe Works

Audiosafe integrates with contact centre platforms to automatically export and archive call recordings and associated metadata into a secure AWS S3 environment. The solution includes:

  • Authentication and Authorisation Module
  • Secrets Management
  • Genesys API Integration
  • Recording Retrieval API
  • Searchable User Interface built on React

Security and Compliance

Audiosafe is designed with security at its core:

  • Encryption at rest using AWS S3 Managed Keys (SSE-S3)
  • Encryption in transit using HTTPS (TLS 1.3)
  • Audit logging of user access and actions
  • GDPR-compliant data deletion process via Maintel support tickets
  • Penetration testing options available

Supported Platforms

Currently supported: Genesys Cloud
Further integrations (e.g., Redbox API) are on the roadmap.

Accessing and Using the Audiosafe UI

The Audiosafe User Interface allows you to:

  • Log in securely with your corporate SSO credentials
  • Search recordings by date, contact, direction, user, and queue
  • Playback recordings directly in the interface
  • Export recordings (roadmap feature)
  • Manage admin users and roles (roadmap feature)

Pricing and Packages Overview

Audiosafe is available as a subscription service with Per User Per Month (PUPM) pricing. Packages include:

  • Basic: 1 Admin User, 1TB storage, search and replay
  • Standard: All Basic features + 5 Admins, 5TB storage, delete and export
  • Advanced: All Standard features + unlimited Admins, 20TB storage, role-based access
  • Custom: Bespoke storage and functionality (Price on request)
